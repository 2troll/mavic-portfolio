v21 Released May 2026
UI/UX PRO MAX · Cinematic Scroll & Hardware 3D Simulations
  • newCinematicScrollDeer — SVG Nara deer silhouette driven by useScroll section progress. useTransform maps scroll 0→1 to horizontal position −18vw→108vw. useSpring({stiffness:100, damping:20}) provides organic deceleration on sudden stops. Vertical gallop bounce (animate y:[0,−5,0,−2,0]) completes the living-creature illusion.
  • newDioramaCastle3D — True CSS 3D diorama with transformStyle:"preserve-3d" and perspective:900px. Three mathematically composed SVG layers at translateZ 20/40/60px: sky+moon+stars, Osaka Castle silhouette with eave arches, sakura foreground trees. useMotionValue → useTransform → useSpring({stiffness:300, damping:30}) maps cursor coordinates to rotateX/Y and per-layer parallax offsets. Challenge overcome: avoiding overflow:hidden on preserve-3d containers (clips 3D children in all browsers).
  • newInteractiveToriiGlow — High-fidelity SVG torii (kasagi, shimagi, nuki, hashira pillars, daiwa rings, nemaki base stones). useAnimation with onHoverStart/End drives a 4-keyframe drop-shadow neon pulse loop while hovered, then springs back to zero shadow (stiffness:300, damping:15).
  • newHeroTextKinetic — Cascade text reveal: splits by words or chars, each token animates from {rotateX:−95, y:60, opacity:0, scale:0.82, filter:blur(8px)} with spring(stiffness:260, damping:18). staggerChildren:0.08 (words) / 0.022 (chars). accentWords[] prop applies brand gradient to specified word indices.
  • design7 tour routes introduced (Osaka Food Circuit / Hidden Kyoto / Kobe Refined / Nara Sacred Forest / Kōyasan Pilgrimage / Osaka×Kyoto Grand Day / Mountain Hiking) with booking.html?route=N WhatsApp redirect.
  • perfReact.lazy() + Suspense isolates all Three.js/R3F — WebGL failure can never black-out the rest of the page.
  • fixRoot cause of black-page crash: FinialPulse and Eave were defined inside CastleModel's render body, giving React unstable component identity on every render and breaking hook rules. Moved to module level.
