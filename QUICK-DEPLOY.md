# 🚀 Quick Deploy - No Script Needed!

## Option A: GitHub Pages (Recommended)

### Step 1: Create Repository (30 seconds)
1. Go to **https://github.com/new**
2. Repository name: `mavic-portfolio`
3. Click **Create repository**
4. Copy the HTTPS URL (e.g., `https://github.com/YOURNAME/mavic-portfolio.git`)

### Step 2: Push Code (Run these commands in portfolio folder)
```bash
git remote add origin https://github.com/YOURNAME/mavic-portfolio.git
git branch -M main
git push -u origin main
```

### Step 3: Enable GitHub Pages (30 seconds)
1. In your GitHub repo, click **Settings**
2. Click **Pages** in left sidebar
3. Under "Source", select **Deploy from a branch**
4. Select **main** branch, **/(root)** folder
5. Click **Save**

### Step 4: Done! 🎉
Your site will be live at:
`https://YOURNAME.github.io/mavic-portfolio`

(Wait 1-2 minutes for deployment)

## Option B: Netlify (Fastest - No GitHub needed)

1. Go to **https://netlify.com**
2. Sign up (free)
3. Drag this `portfolio` folder to the Netlify dashboard
4. Done! Instant live URL
