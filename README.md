# Mavic Portfolio Website

A modern, responsive portfolio website built for Mavic.

## 🚀 Quick Start - Choose Your Method

### Method 1: GitHub Pages (Recommended - Free Custom Domain)

**Step 1:** Create repository
- Go to https://github.com/new
- Name: `mavic-portfolio`
- Click "Create repository"

**Step 2:** Run the automated script
```powershell
# In the portfolio folder, right-click and select "Run with PowerShell"
.\deploy.ps1
```

Or manually:
```bash
git remote add origin https://github.com/YOURNAME/mavic-portfolio.git
git branch -M main
git push -u origin main
```

**Step 3:** Enable GitHub Pages
- Settings → Pages → Deploy from branch → main → Save

**Your URL:** `https://YOURNAME.github.io/mavic-portfolio`

---

### Method 2: Netlify (Fastest - No GitHub needed)

1. Go to https://netlify.com
2. Sign up (free)
3. Drag the `portfolio` folder to the dashboard
4. Done! Instant live URL
